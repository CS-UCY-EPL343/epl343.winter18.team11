Το συστημα μας είναι μια Android εφαρμογή για την εταιρία Emira Pottery.

Η εφαρμογή μας έχει σκοπό να διευκολύνει τόσο τους πελάτες του καταστήματος, όσο και τους εργαζόμενους. 
Ένας πελάτης μπορεί να δημιουργήσει λογαριασμό στην εφαρμογή. Μπορεί να δει πληροφοριες σχετικά με την εταιρία.Επίσης μπορεί να δει τα προϊόντα που προσφέρει το κατάστημα και αν επιθυμεί να αγοράσει. Όταν ο πελάτης επιθυμεί να αγοράσει ένα αγγείο, μπορεί να το τοποθετήσει στο καλάθι του. Μπορεί σε οποιαδήποτε στιγμή να αφαιρέσει τα αντικείμενα από το καλάθι ή να προχωρήσει σε αγορά. Αν επιθυμεί, ο πελάτης μπορεί να κλείσει ραντεβού για μάθημα στην ημερομηνία που θέλει. Ακόμα, μπορεί να βαθμολογήσεις την εμπειρία του σχετικά με το μάθημα αγγειοπλαστικής ή τα προϊόντα τα οποία αγόρασε.
Αν ο χρήστης της εφαρμογής είναι διαχειριστής, μπορεί να αλλάξει πληροφορίες, να προσθέσει ή να αφαιρέσει προϊόντα από τη λίστα με τα προϊόντα. Ακόμα, μπορεί να δει τη λίστα των πελατών.
